## üìñ About The Project

This project represents the digital transformation of **SushkovPro**, moving a traditional offline construction business into the digital space. The goal was not just to display information, but to build a high-performance, interactive platform that establishes trust with B2B partners and generates high-quality leads.

The application leverages the latest frontend technologies (**React 19**, **Tailwind v4**) to ensure maximum performance, SEO optimization, and a premium user experience.

### üéØ Business Value
* **Digital Trust:** Created a professional "digital face" for the company to validate credibility with potential partners.
* **Lead Generation:** Engineered a seamless inquiry flow that converts visitors into potential clients.
* **Brand Positioning:** Differentiated the brand from competitors through modern UI and 3D interactive elements.

## üõ†Ô∏è Tech Stack & Key Features

### Core Architecture
* **Framework:** `React 19` + `Vite` for instant server start and lightning-fast HMR (Hot Module Replacement).
* **Styling:** `Tailwind CSS v4` ‚Äî utilized the latest engine for zero-runtime overhead and rapid UI development.
* **Routing:** `React Router` for seamless SPA navigation.

### UX & Interactivity
* **Animations:** Integrated `Framer Motion` for smooth page transitions, scroll-triggered reveals, and micro-interactions that give the site a "premium" feel.
* **3D Elements:** Experimented with `React Three Fiber` (R3F) to introduce 3D concepts into architectural presentations, pushing the boundaries of standard corporate sites.
* **Project Gallery:** Implemented `Swiper.js` for touch-friendly, responsive image sliders showcasing construction projects.

### Lead Generation System
* **Forms:** Built with `React Hook Form` for efficient rendering and validation.
* **Data Handling:** `Axios` for reliable HTTP requests and submission handling.

## üöÄ Getting Started

### Prerequisites
Make sure you have Node.js (v18 or higher) installed.

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/your-username/sushkov-pro.git](https://github.com/your-username/sushkov-pro.git)
    cd sushkov-pro
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
